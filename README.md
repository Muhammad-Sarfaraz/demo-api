# Demo Project API

Simple API project for testing API links using GET, POST, and PUT requests.

## Features

- Test GET requests
- Test POST requests
- Test PUT requests
- Simple JSON-based mock API
- Easy setup for development and testing

## Files

- `db.json` — Mock database for API testing
- `README.md` — Project documentation

## Purpose

This project is created for testing API endpoints and practicing CRUD operations using a simple JSON structure.

It is useful for frontend and backend developers who need quick API testing during development.

## Usage

Use this project to:

- Fetch data using GET
- Create new data using POST
- Update existing data using PUT

## Example

Run the mock server with your preferred JSON server setup and connect your frontend application for testing.

## Repository

Public repository for development and testing purposes.
